# webassembly

Example repository to test the operation of [webassembly](https://github.com/dcodeIO/webassembly).

## Status

This project is unmaintained and no longer receives updates. The last activity dates from October 2018, and it was only intended as a short-lived example for the upstream [webassembly](https://github.com/dcodeIO/webassembly) toolchain. The repository is being archived. Please refer to the upstream project for current usage and examples.
